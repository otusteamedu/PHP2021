# Домашнее задание №1
## 1. Docker

1. Добавить в файл /etc/hosts строку:
```shell
127.0.0.1   olegsv.local
```

2. В папке Docker Переименовать файл '.env.example' в '.env'.
```shell
mv Docker/.env.example Docker/.env
```

3. Перейти в папку Docker и запустить docker-compose/
```shell
docker-compose up -d
```
